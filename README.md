# SafeAI@UAI 2026 Website

This repository is a simple static webpage: a single `index.html` file with assets in `images/`. Because there is no build step or frontend framework here, the easiest way to view changes locally is to serve the directory with a basic static HTTP server.

## Run locally

From the repository root, run:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000/` in your browser.

After editing `index.html`, save the file and refresh the browser to see your changes.
